# Plan Component

When planning a `shinyshadcn` component, include:

1. Component purpose.
2. R function signature.
3. Generated HTML structure.
4. CSS variables/classes.
5. Accessibility requirements.
6. JavaScript behavior, if any.
7. Tests.
8. Example usage.

Write the plan to `docs/agent-plans/YYYY-MM-DD-component-name.md`.
