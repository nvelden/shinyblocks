# Code

> Shinyblocks function: `block_code()`
> Shadcn reference: documentation code examples on
> <https://ui.shadcn.com/docs/components/textarea>
> Status: Runtime presentational component; Phase 7 spec refreshed
> around shipped variant + header + copy-button contract and the
> built-in lightweight tokenizer.

## States

- **default** — bordered, rounded, background-surface frame with
  preformatted code content.
- **outline variant** — transparent surface, otherwise identical to
  the default frame.
- **line numbers** — left gutter numbers generated by CSS counters,
  aligned to the shadcn docs examples.
- **absolute copy button** — quiet icon button positioned in the
  top-right corner of the code frame.
- **copied feedback** — icon swaps from copy to checkmark with an
  active color transition.
- **header (optional)** — upper bar containing editor dots on the
  left, uppercase language label, and inline copy button.

## R API

| Argument | Purpose |
| --- | --- |
| `code` | Code string to display. Single non-empty character. Required. |
| `language` | Optional language label. Drives the lightweight tokenizer and the header label. |
| `copyable` | Whether to render the copy-to-clipboard button. Defaults to `TRUE`. |
| `line_numbers` | Whether to render the gutter. Defaults to `TRUE`. |
| `header` | Whether to render the editor-style header bar. Defaults to `FALSE` to match the shadcn docs look. |
| `variant` | `"default"` (background surface) or `"outline"` (transparent). |
| `class` / `style` | Extra classes / inline style on the runtime wrapper. |
| `...` | Additional attributes passed through. |

## Token contract

| Visual role | Token |
| --- | --- |
| Container BG (default) | `--muted` |
| Container BG (outline) | transparent |
| Border | `--border` |
| Absolute copy button BG | transparent |
| Absolute copy button hover BG | `--accent` |
| Absolute copy button hover text | `--accent-foreground` |
| Line number text | `--muted-foreground` |
| Text foreground | `--foreground` |
| Language label | `--muted-foreground` |

## Deliberate divergences from shadcn

- **Lightweight highlighter** — shadcn's docs examples are produced by
  the docs pipeline (`rehype-pretty-code` / Shiki). `block_code()`
  preserves the surface, gutter, copy action, and whitespace, with a
  small built-in tokenizer for common R / Python / TSX examples
  instead of bundling Shiki.
- **Optional header extension** — `header = TRUE` adds an editor-style
  bar for dashboard use cases. It is not part of the default shadcn
  docs example.
