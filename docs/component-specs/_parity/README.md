# Parity Baselines

Committed computed-style baselines for the ADR 0016 visual-parity harness.

Current scope:
- `button.json`
- `checkbox.json`
- `select.json`
- `slider.json`
- `switch.json`

These files are generated from the dev-only React reference app under
`parity/` and diffed against the local shinyblocks showcase.
