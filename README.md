# shinyshadcn

An experimental R package for building Shiny dashboards with a shadcn/ui-inspired component system.

The goal is to provide the ergonomics of `shinydashboard`, but with:

- composable dashboard layout primitives;
- accessible, modern components;
- theme tokens that can map cleanly to shadcn/Tailwind-style design language;
- a clear bridge between R `htmltools` APIs and frontend assets.

This repository is currently a planning scaffold for Codex and Claude-assisted development.

## Initial Package Shape

```r
library(shiny)
library(shinyshadcn)

ui <- shadcn_page(
  title = "Analytics",
  sidebar = shadcn_sidebar(
    shadcn_nav_item("Overview", icon = "layout-dashboard", selected = TRUE),
    shadcn_nav_item("Reports", icon = "file-chart-column")
  ),
  header = shadcn_header("Analytics"),
  shadcn_card(
    title = "Revenue",
    value = "$42,100"
  )
)

server <- function(input, output, session) {}

shinyApp(ui, server)
```
